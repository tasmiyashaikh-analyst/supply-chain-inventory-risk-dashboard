# Supply Chain Performance & Inventory Risk Dashboard

## 📌 Project Overview

This project focuses on analyzing supply chain operations using Power BI and SQL to monitor inventory risks, shipment performance, product demand, and revenue trends.

## 🎯 Business Problems

- Low stock products causing stockout risks
- Shipment delays affecting operations
- Limited visibility into inventory performance
- Difficulty tracking top-selling products
- Revenue monitoring challenges

## 🛠 Tools Used

- Power BI
- SQL
- DAX
- Data Modeling

## 📊 Dashboard Features

- KPI Cards
- Revenue Analysis
- Top Product Analysis
- Shipment Status Monitoring
- Monthly Sales Trends
- Inventory Risk Tracking
- Interactive Filters

## 🔍 Key Insights

- Home Appliances generated the highest revenue.
- Dress was the best-selling product.
- 75% of shipments were delivered successfully.
- Low-stock products requiring replenishment were identified.
- Revenue trends were analyzed for better decision-making.
